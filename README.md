## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/TahsinGokalp/stories-web.git
   ```
2. Install Composer packages
   ```sh
   composer install
   ```
3. Install NPM packages
   ```sh
   npm install
   npm run dev
   ```
4. Migrate and Seed
     ```sh
       php artisan migrate --seed
     ```
